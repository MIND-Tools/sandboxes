MIND projects

Before you can work in the MIND, you must create a project to store your source code, makefiles, binaries, and related files. MIND projects are displayed in the MIND Navigator view.

Tip: Nested projects are not supported. Each project must be organized as a discrete entity. Project dependencies are supported by allowing a project to reference other projects that reside in your workspace.

For more information about projects and where they are stored, see:
