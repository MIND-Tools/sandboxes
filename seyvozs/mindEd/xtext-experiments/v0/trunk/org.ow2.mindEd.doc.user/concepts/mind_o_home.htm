Mind Development User Guide

Mind IDE 0.2

The Mind Development Toolkit (MDT) is a collection of Eclipse-based features that provides the capability to create, edit, navigate, build, (and debug) projects that use Fractal as a programming language.

The MDT does not include mindc compilers.
