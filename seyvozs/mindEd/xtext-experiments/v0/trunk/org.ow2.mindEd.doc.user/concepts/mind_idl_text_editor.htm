MindEd - IDL Textual Editor

The MindEd IDL editor aims to provide facility for the textual edition of Mind interfaces and types. It is decomposed in two textual editors which propose the same features described for the ADL editor
.
  • Interface definition, ITF editor
    When you create an interface definition (a file using an .itf extension), it should be openned with the ITF editor. At this time, you benefit from the textual facility for editing it.

  • Type definition, IDT editor
    As described for the ITF editor, the edition .idt files will automatically open the associated editor.
